# NIRART MAISON

## Objetivo

Sistema web para gestão completa de locação de roupas para eventos escolares e clientes individuais.

## Tecnologias

* React
* Vite
* Tailwind CSS
* Supabase
* PostgreSQL
* Supabase Auth
* Supabase Storage
